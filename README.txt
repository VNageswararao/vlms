     # vlms
  Vlms Multi-tenancy

Vlms - the world's open source learning platform

Vlms <http://vlms.co.in> is a learning platform designed to provide
educators, administrators and learners with a single robust, secure and
integrated system to create personalised learning environments.

You can download Vlms <https://github.com/VNageswararao/vlms.git> and run it on your own
web server, ask one of our Vlms Partners <https://vlms.co.in/partners/> to
assist you, or have a VlmsCloud site <https://vlms.co.in/cloud/> set up for
you.

Vlms is widely used around the world by universities, schools, companies and
all manner of organisations and individuals.

Vlms is provided freely as open source software, under the GNU General Public
License <https://docs.vlms.org/dev/License>.

Vlms is written in PHP and JavaScript and uses an SQL database for storing
the data.

See <https://docs.vlms.org> for details of Vlms's many features.
